Image Search Abstraction Layer

Created By Josh Hansen

Welcome! This Node.js app utilizes the Bing Search API and node-bing-api package to return json data from get requests made by users. Recent searches are stored in a mongoDB database hosted on mlab.com. Pagination is also supported.

Search Url Example:

https://striped-vegetarian.glitch.me/api/imagesearch/tacos?offset=2 ---- offset parameter optional

Url to See Previous Searches:

https://striped-vegetarian.glitch.me/api/recentsearches

Type in a URL above to see how it works!
