Image Search Abstraction Layer

Created By Josh Hansen

Welcome! This Node.js app utilizes the Bing Search API and node-bing-api package to return json data from get requests made by users. Users can input URL with an image search value for Bing API to return results on or input URL to show recent searches made by other users. Recent searches are stored in a mongoDB database hosted on mlab.com. Pagination is also supported.

This app was created as part of my FreeCodeCamp Back End Developer tasks. Feel free to check out my github profile to see other FreeCodeCamp and miscallenous projects I've worked on.

Technologies Used: Node.js, MongoDB, Express.

Search url examples that'll return image search for tacos:

https://striped-vegetarian.glitch.me/api/imagesearch/tacos?offset=2 ---- offset parameter optional

Url to See Previous Searches:

https://striped-vegetarian.glitch.me/api/recentsearches
